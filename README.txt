Setup

To set up the application, please follow these steps:

Create a new SQL Server database.
Run the provided table creation script in the newly created database. (Update the script with your DB name)
Run the insert data script to populate the database table with sample data.
Open the appsettings.json file and update the connection string to point to the newly created database instance.
Build the application and ensure it works as expected.
